## Sales & Inventory Registration System (Streamlit App)

- Sales log  
- Dashboards & simple reports  
- Clean, minimal UI  
- PostgreSQL support  

**Repository:** https://github.com/renecano/ventas-inventario-streamlit

## IoT Plant Monitoring System (ESP8266 + MQTT + Flask + SQLite)
An IoT system that monitors temperature and humidity using an **ESP8266 + DHT11**. Data is sent via **MQTT**, stored in **SQLite**, and displayed on a real-time dashboard.

**Highlights**
- ESP8266 firmware publishing JSON via MQTT  
- Python backend subscriber  
- Local SQLite storage  
- Real-time dashboard (Flask + Bootstrap + Chart.js)  
- Modular and scalable structure  

**Repository:** https://github.com/renecano/IoT-Plant-Monitoring-System-ESP8266-MQTT-Flask-SQLite

## Fork/Join DAG Visualizer (Java + JavaFX)
An interactive tool that loads **DAGs**, renders them in **JavaFX**, and automatically generates **Fork/Join parallel code**, including execution simulation and validation.

**Highlights**
- Load DAGs from `.txt` adjacency matrices  
- BFS-based level layout  
- JavaFX rendering with animations  
- Fork/Join code generation  
- Parallel execution simulation with branch coloring  
- Structural validation (cycles, disconnected nodes, incorrect joins)

**Repository:** https://github.com/renecano/grafos-fork-join-visualizer
